# Multi-Tenant SaaS Backend API

A production-style backend for a SaaS product with multi-tenancy, authentication, and role-based access control.

## Features

- **Multi-Tenancy**: Each organization has isolated data
- **JWT Authentication**: Secure token-based authentication
- **Role-Based Access Control**: Admin and member roles
- **SQLite Database**: Simple file-based database with full control
- **REST API**: Clean, well-structured endpoints
- **Data Isolation**: Middleware ensures tenant data separation
